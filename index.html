📚 Meta Ads Basics

Meta Advertising ka foundation — Facebook, Instagram, WhatsApp par advertising ka complete overview. Bilkul beginners ke liye.

Intro
Ad Structure
Objectives
Placements

Meta Advertising kya hai?

Meta Ads = Facebook + Instagram + Messenger + WhatsApp + Audience Network par advertising. 3.3 billion daily active users tak ek platform se pahuncho.

  • Facebook: Sabse bada platform — News Feed, Groups, Marketplace
  • Instagram: Visual-first — Feed, Stories, Reels — young urban audience
  • Messenger: Direct message ads — high intent users ke liye
  • WhatsApp: Click-to-WhatsApp — India mein 90%+ penetration, extremely powerful
  • Audience Network: Partner apps mein ads — cheapest CPM
💡 India Tip

Click-to-WhatsApp ads India mein bahut powerful hain. CPL typically 60-70% lower hota hai website leads se. Zaroor try karo!

3-Level Structure

1
Campaign Level — Objective
Kya achieve karna chahte ho decide karo: Awareness, Traffic, Leads, ya Sales. Campaign Budget Optimization (CBO) yahan ON karo.
2
Ad Set Level — Audience + Placement + Budget
Kaun dekhega (audience), kahan (placement), kitna budget, kab (schedule). Ek campaign mein multiple ad sets ho sakte hain.
3
Ad Level — Creative
Actual ad — image, video, text, CTA button. Ek ad set mein 3-5 ads rakho — Meta best performing ko identify karega.
⚠ Common Mistake

Sirf 1 ad per ad set mat rakho. Hamesha 3-5 creative variations test karo. Meta automatically best ko zyada dikhata hai.

Campaign Objectives

ObjectiveBest For2026 Note
AwarenessBrand buildingReels ke liye best
TrafficWebsite visitorsCAPI ke saath use karo
LeadsPhone/email collectWhatsApp integration
SaleseCommerce purchasesAdvantage+ Shopping
✓ Beginner Tip

Data kam hai? Pehle Traffic objective se data collect karo, phir Sales objective pe shift karo jab pixel optimized ho.

Ad Placements 2026

  • Facebook Feed — highest reach, sab age groups
  • Instagram Feed — young audience, high engagement
  • Stories — full screen 9:16, high swipe-up rates
  • Reels — fastest growing 2026, best ROI currently
  • Facebook Marketplace — shopping intent users
  • Audience Network — cheapest CPM, awareness
💡 2026 Strategy

Pehle Advantage+ Placements (automatic) use karo. Reels placements consistently best ROI de rahe hain — nayi creatives 9:16 format mein banao.

⚙ Account Setup — Step by Step

Business Manager se Ad Account tak — sab kuch properly configure karo. Yeh foundation hai aapki poori advertising ki.

Business Manager Setup

1
business.facebook.com par jaao
Personal Facebook account se login karo. "Create Account" click karo. Business name, naam, aur business email daalo.
2
Ad Account Create karo
Business Settings → Accounts → Ad Accounts → Add → Create New. Currency: INR. Time Zone: Asia/Kolkata (IST).
3
Payment Method Add karo
Billing settings → Add payment method. Credit card, debit card, ya UPI add karo. ₹500 se start karo. Prepaid billing safe hai beginners ke liye.
4
Facebook Page Connect karo
Business Settings → Pages → Add Page. Facebook Page connect karo. Instagram Account bhi yahi se link karo.
5
Two-Factor Authentication ON karo
Settings → Security → 2FA → Enable. SMS ya Authenticator App use karo — mandatory for security!
⚠ Restriction Se Bachne Ke Liye

Pehle hafta ₹200-500/day se shuru karo. Real business information use karo. Fake claims kabhi mat dalo — permanent ban ho sakta hai.

🔵 Meta Pixel & Conversions API

Website tracking ka foundation. Pixel browser se track karta hai, CAPI server se. Dono saath use karo 2026 mein.

💡 2026 Critical

iOS 18 aur browser privacy changes se pixel data 30-40% loss hota hai. CAPI (server-side tracking) mandatory hai accurate data ke liye!

Pixel Setup

1
Events Manager mein jaao
business.facebook.com → Events Manager → Connect Data Sources → Web → Meta Pixel → Create Pixel.
2
Install karo
WordPress: Meta Pixel plugin. Shopify: Facebook & Instagram app. WooCommerce: PixelYourSite plugin. Custom: head tag mein paste karo.
3
Standard Events Track karo
ViewContent, AddToCart, InitiateCheckout, Purchase — developer se add karvao. Har event mein value aur currency INR bhejo.
4
Pixel Helper se Verify karo
Chrome mein "Meta Pixel Helper" extension install karo. Green checkmark = pixel sahi kaam kar raha hai.

🚀 Campaign Kaise Create Karein

Ads Manager mein pehla campaign step-by-step. Har cheej explain ki gayi hai.

1. Campaign
2. Ad Set
3. Creative
4. Publish

Campaign Level

1
Ads Manager Open karo
facebook.com/adsmanager → green "+ Create" button click karo.
2
Objective Select karo
Website traffic → Traffic. Phone/email → Leads. Product bechna → Sales. Brand awareness → Awareness.
3
Campaign Name do
Example: "Kurta_Sales_Jan2026_v1" — clear naam rakho dhundhna aasaan ho.
4
Advantage Campaign Budget (CBO) ON karo
Toggle ON karo. Daily budget daalo — minimum ₹500/day. Meta khud best ad set ko budget dega.
💡 Budget Tip

7 din ka total budget pehle decide karo. Example: 7 din × ₹500 = ₹3,500. Pehle hafte sirf data dekho — koi change mat karo.

Ad Set — Audience & Placement

1
Conversion Location Set karo
Sales ke liye: Website select karo. Apna Pixel choose karo. Optimization Event: Purchase ya Landing Page Views.
2
Audience Define karo
Location: India ya specific city/state. Age: product ke hisaab se. Interests: relevant keywords. Audience size: 1-10 million ideal.
3
Advantage+ Audience try karo
2026 recommended — Meta khud audience decide karta hai. Sirf location aur age daalo, baaki Meta pe chhod do.
4
Placements: Advantage+ Rakho
Automatic placements ON rakho. Meta best placement find karega. Manual placement baad mein test karo.

Ad Creative Banao

1
Format Choose karo
Single Image: simple, fastest. Video: best CTR. Pehli baar ke liye Single Image ya Short Video recommended.
2
Creative Upload karo
Image: 1080x1080px, JPG/PNG, max 30MB. Video: 1080x1080 ya 9:16, MP4. 3-5 different creatives upload karo ek ad set mein.
3
Ad Copy likhna
Primary Text: 125 chars. Headline: 40 chars. CTA button choose karo: "Shop Now", "Send Message", etc.
4
Destination URL add karo
Landing page URL daalo. Mobile pe fast load hona chahiye. UTM parameters add karo tracking ke liye.
✓ Creative Tips 2026

First 3 seconds mein hook. Subtitles hamesha add karo (85% log mute mein dekhte hain). Human faces wale ads better perform karte hain.

Review & Publish

1
Preview dekho
Ad Preview mein dekho — Feed, Stories, Reels sab mein. Mobile preview zaroor check karo.
2
Publish karo
"Publish" click karo. Meta review 24-48 hours mein karta hai. Approve hone par email aayega.
3
First 7 Days — Hands Off!
Learning phase mein campaign touch mat karo. Meta ko 50 optimization events collect karne do. Sirf data dekho!
⚠ Learning Phase

Pehle 7 din mein koi bhi change (budget, audience, creative) learning phase restart kar deta hai aur data waste hota hai. Patience rakho!

🎯 Audience Targeting — Complete Guide

Sahi audience = campaign ka 50% success. Teen types — cold, custom, aur lookalike.

Cold Audience

  • Interest Targeting: Hobbies, pages liked. 5-15 interests ek ad set mein stack karo.
  • Demographic: Age, gender, education, job title, life events.
  • Broad/Advantage+ (Best 2026): Age + gender sirf do, baaki Meta AI pe — consistent winners deta hai.

India Targeting

SegmentHow to TargetBudget

Custom Audiences

1
Website Custom Audience
"All website visitors last 30 days" banao. "Add to Cart but no Purchase" — sabse hot retargeting audience. "Purchasers" — upsell ya exclude.
2
Video Engagement Audience
Video ka 75% ya 95% dekha jinhone — high intent. Inhe retargeting ads dikhao.
3
Customer List Upload
Phone numbers ya emails upload karo (minimum 1000, ideally 10,000+). 40-60% match rate India mein.

Lookalike Audiences

Meta ko kehna: "mere existing customers jaise aur log dhundo." Sabse powerful scaling tool.

  • Best source = Purchase audience (actual buyers). Minimum 100 logon ka source chahiye.
  • 1%: Sabse similar, quality zyada — main campaigns ke liye best.
  • 2-5%: Quality aur reach ka balance — scaling ke liye.
  • 6-10%: Zyada reach, lower quality — awareness ke liye.
💡 LAL Strategy

3 ad sets banao: 1% LAL, 2-3% LAL, 4-5% LAL. Alag budgets. India mein 1% LAL usually 1-2 million audience hoti hai.

Winning Creative Checklist

  • First 3 seconds mein scroll-stopping hook — question, bold statement
  • Captions/subtitles hamesha add karo — 85% log sound off mein dekhte hain
  • Mobile-first design — small screens pe readable text
  • Human faces dikhao — emotional connection create hota hai
  • UGC style organic content better perform karta hai
  • Ek hi clear CTA — user ko confuse mat karo
✓ Testing Framework

Ek ad set mein 5 creatives: 2 image, 2 video, 1 carousel. 7 din baad worst performers turn off karo. Winners ko scaling campaign mein higher budget ke saath move karo.

📈 Scaling Strategies — 10x Results

Jab campaign kaam kar raha ho — tab scale karo. Galat time par scale karna paisa waste hai.

2 Types of Scaling

↔
Horizontal
Naye audiences add karo. Safer method.
↕
Vertical
Existing winners ka budget badao. Max 20-30% per 3-4 days.

Budget Kab Badhayein

  • ROAS target se 1.5x consistently 7+ din → budget 20% badao
  • CPA target se 30% neeche → budget 30% badao
  • Winning creative identify → duplicate ad set with higher budget
  • Frequency 3+ → pehle nayi creatives add karo, tab scale
💡 Rule of Thumb

Budget change ke baad 3-4 din wait karo. Roz roz change karna learning phase disrupt karta hai.

🛒 eCommerce Ads — Online Store Grow Karo

eCommerce ke liye Meta Ads ka complete playbook — catalog, dynamic ads, Advantage+ Shopping.

eCommerce Setup

1
Product Catalog Upload karo
Commerce Manager → Catalog → Add Products. Shopify/WooCommerce direct integration available hai.
2
Advantage+ Shopping Campaign
Objective: Sales → Advantage+ Shopping. Budget: minimum ₹1000/day. 5+ images/videos provide karo.
3
Dynamic Product Ads (DPA)
Visitor ne jo product dekha — wohi retargeting mein dikhao. "Cart abandonment" most important segment hai.
4
ROAS Target Calculate karo
Break-even ROAS = 1 ÷ Gross Margin%. Example: 30% margin → 3.33 minimum ROAS. Target 4x+ rakho.
